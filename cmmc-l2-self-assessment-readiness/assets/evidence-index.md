## Redaction Guidelines for Customer Deliverable

If providing a redacted evidence index to the customer, remove or obscure:
- Internal system IP addresses and hostnames
- Employee names (replace with roles)
- Specific software versions that could aid exploitation
- Internal network topology details
- Proprietary technical implementation specifics

Retain sufficient detail to demonstrate control implementation without exposing operational security information.

## Cross-Reference to Control Assessment

Each control in the control-assessment-workbook.csv with Status = "Met" must reference at least one Evidence ID from this index in the "Evidence Reference" column. Controls without evidence references cannot be marked as "Met".
